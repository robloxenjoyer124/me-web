# iqad

minimal personal site with live discord presence.

## features

- **starfield background** — twinkling stars with random shooting stars
- **live presence** — real-time discord status & spotify via [lanyard](https://github.com/Phineas/lanyard) websocket
- **clean design** — monochrome, no frameworks, just html/css/js

## stack

`html` · `css` · `javascript` · `lanyard api`
